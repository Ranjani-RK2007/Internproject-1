# Internproject-1

Medi-appoin is a mobile application designed to simplify the process of booking doctor appointments. The app allows users to register, log in, browse different medical specialties, select doctors, and schedule appointments conveniently from their smartphones.
The application provides a user-friendly interface where patients can choose from various doctor categories such as Orthopedic, Cardiologist, Eye Specialist, General Physician, Pediatrician, Dermatologist, and more. Users can view doctor details, select available time slots, and confirm appointments instantly.
The main objective of Medi Appoin is to reduce waiting time, improve healthcare accessibility, and provide a seamless appointment management experience for patients and healthcare providers.
